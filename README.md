# ProyectoBD

Este proyecto utiliza JavaFX para la interfaz gráfica. Para que en el menú principal se muestre el logotipo de la aplicación, coloque la imagen `logo.png` en la ruta:

```
src/proyectobd/images/logo.png
```

Al iniciar la aplicación, la pantalla principal cargará dicha imagen como fondo y mostrará un mensaje de bienvenida.
